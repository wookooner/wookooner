📌 Project Roadmap
Personal Digital Trace Manager (PDTM)

“내 디지털 흔적을 내가 통제하는 시스템”

0. 프로젝트 핵심 정의 (변하지 않는 원칙)
🎯 미션

사용자가 자신의 인터넷 활동으로 남는 흔적을 인지·관리·통제할 수 있도록 돕는
Local-first, Privacy-first 디지털 관리 시스템을 만든다.

🔐 절대 원칙 (Non-Negotiable)

Local-First

사용자 데이터는 기본적으로 로컬에만 존재

Zero Knowledge

개발자/서버는 사용자 데이터 접근 불가

No Raw Sensitive Data Storage

쿠키 값, 세션 토큰, 입력 폼 내용 저장 금지

Single Source of Truth (SSOT)

판단/정리는 Service Worker가 담당

Explainable 판단

“왜 위험한지 / 왜 분류됐는지” 설명 가능해야 함
